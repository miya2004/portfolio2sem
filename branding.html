Key Features

Overall, the analysis helped me create the website prototype for the client. I chose to focus on both tech inspirations and healthcare, as my assignment was to develop branding. I didn't want to alter their structure of studying, so all the information in the prototype comes from Fonty's website and Bart's Hospital.

I have circled the elements I used in my prototype in red. For more detailed information, you can refer to my PDF document.

Initial Design

The landing page initially featured an egg inside a CD as an interactive "easter egg". Clicking the egg would reveal three options for potential discounts. This design was later changed based on feedback to connect more closely with the band.

Iterative Process

The second version included a "My Tickets" button, which was later removed as unnecessary for a single-event website. By the third version, I integrated a red CD with a portal leading to ticket purchases.

Key Features

The final version addressed feedback about distracting spinning CDs and lack of event information. Spinning CDs remained but redesigned to focus user attention. A "Learn More" button was added to direct users to detailed event information.

Additional Elements

Navigation was simplified, replacing the clickable content table with an informative image. The countdown feature was incorporated alongside a prominent "Get Your Tickets" button. Social media integration evolved from a static Instagram image to a 3D gallery room concept created in Blender.
